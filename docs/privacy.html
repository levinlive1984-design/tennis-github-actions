Privacy Policy

Last updated: 2026-06-30

This GPT Action reads a public JSON file hosted on GitHub Pages: today_matches.json. The JSON contains tennis match information filtered by an automated Python workflow, including match time, league, players, odds, and source.

Data collected

This Action does not collect, store, sell, or share personal information from users. It only retrieves publicly available tennis match data from the GitHub Pages endpoint.

Use of data

The retrieved match data is used only to help the GPT analyze tennis BO3 Over 2.5 Sets candidates.

Third-party services

The data endpoint is hosted on GitHub Pages. When the GPT Action is used, a request may be sent to GitHub Pages to retrieve the JSON data.

For questions, please contact the GPT owner.
